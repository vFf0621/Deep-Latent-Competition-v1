## Installation

To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```
To install the Multi-Car Racing environment, please copy the multi_car_racing folder into the home directory. 
And then run:
```
cd multi_car_racing
pip install -e .
```

## run

### Dreamer
